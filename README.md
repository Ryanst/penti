# penti
喷嚏网 Android 客户端

- RxJava+Retrofit+Okhttp完成自定义网络请求
- Gson格式自定义处理
- RxView点击事件处理
- Glide图片异步加载和缓存
- MVVM架构：Databinding实现
- RecyclerView加载动画和加载更多
- Blockcanary检测内存泄露
- Metrial Degisn 风格设计
